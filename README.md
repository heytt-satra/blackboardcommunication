# Blackboard Communications

Brand Experience Agency website built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- **Modern Design**: Dark theme with vibrant orange accents
- **Responsive**: Fully optimized for mobile and desktop
- **Performance**: PWA-ready capabilities (currently disabled), optimized 3D assets
- **Interactive**: Custom sliders, card swaps, and WebGL animations

## Tech Stack
- Next.js 14
- Tailwind CSS
- Framer Motion
- React Three Fiber (Drei)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
